Excel Functions and statistical analysis

In this project, a range of Excel functions, including XLOOKUP, VLOOKUP, INDEX & MATCH, and various aggregation formulas, were employed within an Excel worksheet to analyze the dataset. These tools were instrumental in dissecting the data, allowing for a comprehensive examination and a glimpse into insightful trends. By leveraging these functions, the analysis could efficiently reveal patterns and correlations, providing valuable insights into the underlying dataset. This method not only optimized the data exploration process but also enhanced the clarity and depth of the insights gained.

HR Dataset Analysis (Power Query)

In this project, the HR dataset was analyzed using Power Query for data transformation, converting the dataset into a structured table format. Additionally, Pivot Tables were utilized to facilitate a deeper exploration of the data, aiding in the creation of an insightful dashboard. This dashboard serves as a visual tool to understand the diverse backgrounds of the company's employees, highlighting key trends and patterns that are crucial for strategic decision-making. This approach not only simplifies the analysis but also provides a clear, actionable overview of employee demographics and their implications for the company.
